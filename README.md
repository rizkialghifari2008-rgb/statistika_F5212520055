# statistika_F5212520055
Proyek ini menggunakan regresi linear berganda untuk mengestimasi produksi jagung berdasarkan luas panen dan produktivitas dari file data Anda. Fungsinya adalah mengukur kontribusi tiap variabel guna memprediksi hasil panen secara akurat, sehingga membantu dinas terkait menyusun strategi pangan dan alokasi sumber daya pertanian di masa depan.
